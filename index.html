hello world # 깃 커밋 방법 0. git config --list git config user.name "계정아이디" git config user.email "계정이메일" 1. 로그인 확인 (계정명 확인) : gh auth status 2. 주의 : 프로젝트 작업 전 : git pull 3. 프로젝트 작업 후 ( 디렉토리 추가 / 파일 추가 / 수정 / 삭제 .... ) 3. 변경된 대상 파일들을 모두 스테이징 : git add --all 4. commit ( 변경 내역을 제 3자가 알아볼 수 있는 규칙으로 메세지를 입력 ) Create , Add , Delete, Update .... git commit -m "메세지" 5. push ( 로컬프로젝트를 -> 원격 저장소에 반영) git push -u origin main
